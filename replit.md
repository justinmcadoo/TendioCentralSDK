# TendioCentral SDK

## Overview

This is the **TendioCentral SSO SDK** — a TypeScript library that provides OAuth 2.0/OIDC authentication, webhook verification, and user management for Tendio-integrated applications.

This is a **pure library** (not a web application). It is published to npm (GitHub Package Registry) and consumed by other Node.js/TypeScript projects.

## Project Structure

```
src/
  core/
    client.ts       - Main TendioAuth client class
    config.ts       - App config fetching
    jwks.ts         - JWKS key fetching and caching
    management.ts   - User management API
    pkce.ts         - PKCE helpers (code verifier, challenge, state)
    tokens.ts       - Token exchange, refresh, revocation
    webhook.ts      - Webhook signature verification
  express/
    index.ts        - Express integration entry point
    middleware.ts   - Express middleware (requireAuth, etc.)
  index.ts          - Main library entry point
  types.ts          - All TypeScript types and interfaces
  __tests__/
    appendix-b.test.ts  - Tests
dist/               - Compiled output (generated by tsc)
```

## Build & Development

- **Language:** TypeScript (compiled to ESNext)
- **Package Manager:** npm
- **Build:** `npm run build` (runs `tsc`)
- **Type Check:** `npm run typecheck`

## Dependencies

- `jose` — JWT/JOSE operations for token verification
- `express` (peer, optional) — Express.js integration

## Publishing

Published to GitHub Package Registry (`https://npm.pkg.github.com`) as `@justinmcadoo/tendiocentral-sdk`.
